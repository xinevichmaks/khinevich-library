# Khinevich Library — кабинет репетитора

Полная синхронизация с макетом. Запуск:
```
npm install
npm run dev
```

## Что внутри
- **Роли**: Репетитор, Ученик, Родитель — с синхронизацией в реальном времени через Firestore
- **Главная**: свечной график динамики баллов (наведение — подсказка с баллом), 2 календаря-виджета (занятия — красным, пробники — зелёным), «Слабые места» по темам вопросов, до 5 последних домашних заданий; у родителя — счётчик оплаченных занятий + прогноз даты новой оплаты
- **Расписание**: статусы занятий (Запланировано / Проведено / Перенесено / Отменено) с сохранением истории переноса, счётчик оплаченных занятий (+1/−1)
- **Домашка**: выдача сразу нескольким ученикам, необязательные вопросы с автопроверкой (тема вопроса → «Слабые места»), статус выбирает репетитор, просмотр как ученик выполнил
- **Пробники** (отдельный раздел): 4 типа вопросов — один правильный ответ (2–6 вариантов), текстовый ответ, установление соответствия (формат ЕГЭ), графиковый (текст + картинка, проверяется вручную репетитором с выставлением баллов)
- **Библиотека**: свои теги, палитра из 15 цветов, переименование/удаление
- **Журнал оценок**: формат ЭЖД, смешанная шкала (5→100%, 4→80%, 3→60%, 2→40%, либо балл из максимума), учитывает домашку, пробники и ручные оценки
- **Регистрация ученика**: выбор предмета (Обществознание / Математика / Английский язык / Поступление за границу)
- Библиотека, Полезное, Доска, Связь, Чат — без изменений

## Важно про безопасность
Правила Firestore пока в тестовом режиме — перед реальным запуском с учениками их нужно ужесточить.

## Что пока не перенесено из макета
1. Лента уведомлений и история платежей у родителя (в макете это были мок-данные без реальной Firestore-коллекции) — при необходимости соберу.
2. Реальные видеозвонки между устройствами — нужен WebRTC-сигналинг.
3. Реальная привязка Google Calendar + пуш-напоминания.
4. Изображения в графиковых вопросах пробников хранятся как base64 прямо в Firestore-документе — работает, но для больших/многих фото лучше подключить Firebase Storage (сейчас не подключён).
